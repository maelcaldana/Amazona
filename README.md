# Robot with Selenium2Library

Small example using robot framework on Amazon.com

### Prerequisites

* Python 2.7
* Pycharm (32bits version)
* pip (9.0.1) with:
    * decorator (4.2.1)
    * robotframework (3.0.2)
    * robotframework-selenium2library (1.8.0)
    * selenium (3.9.0)
    * setuptools (28.8.0)


### Running

```
pybot -d results Amazon.robot
```
